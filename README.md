# 🧠 Quiz Master: Pensée Critique & Audace

Bienvenue dans **Quiz Master**, une expérience interactive conçue pour tester et affiner vos soft-skills, en particulier votre **pensée critique** et votre **audace**.

![Aperçu du Projet] (Une interface moderne et immersive avec des effets de flou et un fond dynamique)

## ✨ Caractéristiques

- 🌊 **Fond WebGL Immersif** : Un arrière-plan animé généré par des shaders GLSL personnalisés pour une ambiance "Cyber-Zen".
- 💎 **Design Glassmorphique** : Une interface moderne utilisant des effets de flou (backdrop-blur) et de transparence pour un look premium.
- 🎵 **Audio Procédural** : Sons de réussite et d'échec générés en temps réel via l'API Web Audio (pas de fichiers MP3 lourds !).
- 📝 **Contenu Éducatif** : 10 questions soigneusement rédigées avec des explications détaillées pour apprendre en s'amusant.
- 📱 **Totalement Responsive** : Profitez de l'expérience sur mobile, tablette ou ordinateur.

## 🚀 Technologies Utilisées

Ce projet a été bâti avec les technologies web les plus modernes :

- **React + Vite** : Pour une rapidité de développement et des performances optimales.
- **Tailwind CSS** : Pour un stylage élégant et rapide.
- **WebGL / GLSL** : Pour le fond animé haute performance.
- **Web Audio API** : Pour la synthèse sonore dynamique.

## 🛠️ Installation et Lancement

Si vous voulez tester le projet localement :

1.  **Clonez le dépôt** :
    ```bash
    git clone [votre-repo-url]
    ```
2.  **Installez les dépendances** :
    ```bash
    npm install
    ```
3.  **Lancez le serveur de développement** :
    ```bash
    npm run dev
    ```
4.  Ouvrez votre navigateur sur `http://localhost:5173`.
